    CSS
    Cascading Style Sheets, folha de estilo ao código criado por linguagense funções em alterar elementos como os do texto, fonte e espaçamento entre blocos.
    Javascript(JS)
    Linguagem de criação de scripts ou uma linguagem interpretada, implementando itens complexos em páginas, baseadas em objetos.
    Trata-se de um programa como um conjunto de partes individuais, criação de aplicações web, sites, aplicativos e sistemas online, permite inserir efeitos fazendo com que fiquem mais dinâmicos e permite ao programador da web melhorar a página do site, além da interatividade.
